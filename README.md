# Real-Estate-Dashboard---Power-Bi
Interactive Real Estate ROI Dashboard built in Power BI 

Analyzes a portfolio of 100 commercial/residential properties across major U.S. cities (Denver, New York, Miami, Chicago, Houston, Los Angeles, etc.) Data sources: three cleaned CSV files

Data sources: three cleaned CSV files

Files :

Income Expenses - It has - PropertyID ||	Date || RentalIncome || Expenses

Leases - It has - PropertyID || Tenant || LeaseStart || LeaseEnd  || Occupancy

Properties - It has - PropertyID || PropertyName || City	Type || PurchasePrice || MarketValue
